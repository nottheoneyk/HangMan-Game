## Hangman Game 
A simple and interactive command-line Hangman game built using Python.  
This project demonstrates basic programming concepts like loops, conditionals, and string handling.

### Features
- Random word selection
- Limited number of attempts
- Input validation
- User-friendly prompts

### Tech Stack
- Python

### What I Learned
- Working with loops and conditions
- Handling user input in Python
- Building a simple game logic

### Future Improvements
- Add GUI using Tkinter

- Add word categories
